## About this Project

A Node-based README generator

### Built with

- Commander

## Getting Started

### Prerequisites

- Node

### Installation

1. Install the dependencies.

```
npm i
```

2. Run the build tool.

```
npm run dev
```

## Deployment

Inside this project directory, run `npm i -g .` to install this local package globally.

## Usage

Run `readme create`.
